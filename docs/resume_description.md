# Resume-Ready Project Description

**AI Study Companion (GenAI RAG System)**

- Developed a full-stack web application enabling students to upload study materials and interact with an AI-powered tutoring assistant.
- Implemented end-to-end GenAI pipeline covering document ingestion, NLP-based chunking, embedding generation, vector storage (ChromaDB/FAISS), semantic search, and LLM-driven answer synthesis.
- Built RESTful backend using Node.js/Express with MongoDB for user/document management and integrated OpenAI's APIs for embeddings and generation.
- Designed React frontend featuring authentication, file upload, chat-based Q&A, dynamic summarization, and quiz generation.
- Ensured modular architecture facilitating scalability, reusability, and maintainability; secured endpoints with JWT-based auth.
- Produced comprehensive documentation for architecture, data flow, API design, and implemented prompt engineering strategies.
- Tech stack: React, Node.js, Express, MongoDB, FAISS/ChromaDB, OpenAI, Docker (optional), REST APIs.

This capstone project demonstrates expertise in AI-driven application design, full-stack development, and modern GenAI techniques.
